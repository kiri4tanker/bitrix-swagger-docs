# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.1.x | yes |
| 1.0.x | no |

## Reporting a Vulnerability

If you find a security issue, please do not open a public issue first.

Send report to: `kiri4tankerofficial@gmail.com`

Include:
- affected version (`install/version.php`)
- minimal reproduction steps
- expected vs actual behavior
- impact estimate

Response targets:
- initial response: within 3 business days
- mitigation plan: within 7 business days

After fix is released, public disclosure is welcome.
